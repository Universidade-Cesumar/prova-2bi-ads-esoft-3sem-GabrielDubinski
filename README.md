Projeto de controle de almoxarifado com interface web simples.

O sistema permite cadastrar materiais com nome e quantidade. Os dados são mantidos em um array no JavaScript, e a lista de materiais é atualizada dinamicamente na página.

Também foi feita uma melhoria visual na interface para deixar o cadastro e a tabela mais fáceis de usar.

Arquivos principais:
- `index.html`: estrutura da página.
- `styles.css`: estilo visual.
- `main.js`: lógica de cadastro, exibição e exclusão usando array.
